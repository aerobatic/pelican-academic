Introduction
==================

Pelican Academic is a theme and tooklit for the `Pelican static site generator <http://docs.getpelican.com/en/stable/>`_ specifically designed for academics and researchers to easily build a modern and attractive online professional portfolio and blog. Python based technologies are chosen due it's widespread use within the academic and research communitity. Pelican Academic is highly inspired by `Hugo Academic <https://sourcethemes.com/academic/>`_ by `George Gushen <https://twitter.com/georgecushen>`_.

See the demo site at `<https://pelican-academic.aerobaticapp.com>`_.

Highlights
==================

- Create a new personal academic blog / professional portfolio site in 5 minutes or less.
- Supports deep customization via `Jinja2 <http://jinja.pocoo.org/docs/2.10/>`_ template inheritance and re-usable macro components.
- Smooth rendering of mathematical formulae using `MathJAX <https://www.mathjax.org/>`_.
- Modern responsive design that looks good on all devices
- Author different categories of articles including blog posts, publications, and projects.
- Conforms to the latest `JSON-LD <https://json-ld.org/>`_ metadata standard for discoverability via search engines and social media.
- CSS based on the latest `Bootstrap 4 <https://getbootstrap.com/>`_.
- Lightweight design with minimal library bloat for fast loading on even low bandwidth connections
